**URL:** https://www.usgs.gov/centers/national-minerals-information-center/mineral-commodity-summaries

**Download:**
```

mcs2025-lithium.pdf
```

**Data to extract per material per year:**
- Production by country (tonnes)
- Top-3 producer share (%)
- Calculate HHI = Σ(share_i²) × 10,000

## HHI Reference Guide

The Herfindahl-Hirschman Index measures market concentration:

```python
def calculate_hhi(production_shares: dict) -> float:
    """
    Calculate HHI from production shares.

    Args:
        production_shares: dict of {country: share_percentage}
        Example: {"Indonesia": 67, "Philippines": 9, "Russia": 7, "Other": 17}

    Returns:
        HHI value (0 to 10,000)
        < 1,500: Competitive market
        1,500–2,500: Moderately concentrated
        > 2,500: Highly concentrated (supply chain risk)
    """
    return sum(share**2 for share in production_shares.values())
```
